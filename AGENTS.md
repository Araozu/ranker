# Project

## Stack

- Sveltekit with Svelte 5, full stack SSR mode
- Use runes, DO NOT use legacy dollar syntax "$ : stuff"
- SQLite with Drizzle
- Tailwind & Shadcn Svelte for UI components
- Prompt the user for installing new shadcn components if needed
- Always use pnpm
- Always use typescript & lang="ts"
